# Closed Work

Closed active packets are preserved in descriptive directories with one of these dispositions: `Completed`, `Deferred`, `Cancelled`, or `Superseded`. Each directory contains:

- `PL-YYYYMMDDTHHMMSSZ.sdd.md` — the final SDD-formatted active plan at time of closure.
- `PL-YYYYMMDDTHHMMSSZ.backlog.md` — the original backlog entry, if the plan was activated from backlog.
