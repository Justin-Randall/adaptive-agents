# Memory

This directory is for promoted, durable lessons and preferences that apply across projects.

Add content here only after a retrospective or repeated observation has been reviewed and promoted into reusable guidance. Raw session notes belong in [retrospectives/inbox](../retrospectives/inbox/README.md), not here.

Keep memory entries short, evidence-backed, and easy for agents to route. If a lesson becomes procedural, promote it into an instruction, skill, or playbook instead of expanding memory indefinitely.
