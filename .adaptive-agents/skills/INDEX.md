# Project Skills

This directory contains task-specific workflows owned by Adaptive Agents.

- [Manage planning](manage-planning/SKILL.md): use when reading or changing active, backlog, or closed project work.
- [Manage retrospectives](manage-retrospectives/SKILL.md): use when capturing or promoting behavior changes that may be project-specific.

Add another focused `<skill-name>/SKILL.md` only when its workflow is specific to this project, and link it from this index. Reusable cross-project skills belong in the canonical Adaptive Agents repository.
