# Adaptive Agents Project Layer

This hidden directory contains project-specific agent guidance, skills, memory, retrospectives, playbooks, and planning. It is owned by this project and may be edited like other project documentation.

[INDEX.md](INDEX.md) is the authoritative routing source. Start there to inspect current work, backlog plans, local guidance, and retrospective learning.

Capture behavior intended only for this project in the local retrospective inbox. Escalate a lesson to user-wide Adaptive Agents only through a separate, sanitized, explicitly approved proposal supported by broader intent or evidence.

Keep all Markdown documents linked from an owning index. Run the linked validator after structural edits so broken links and orphan documents are caught early.
