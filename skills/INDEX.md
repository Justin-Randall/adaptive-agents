# Skills

This directory contains task-specific workflows and domain knowledge that agents should load on demand.

Use a skill when a task has enough specialized context or repeatable procedure that it should not live in global instructions. Keep each skill scoped to a clear trigger and point to authoritative files instead of copying large sections of guidance.
